## 卡片管理：（卡片可以上下拖拽移动改变顺序并提供删除按钮）

# 	人才与对应科研管理
##  人才管理（提供按钮实现录入学者个人简历、高清照片，支持照片裁剪，卡片背景可展示录入的个人高清照片【默认提供增删改查功能】，同时支持此页面选择选项与已录入的资料进行关联（通过下拉选择已存在的论文、课题、专利。选择并保存后，数据库自动建立人才 — 成果关联关系。不允许在人才页面直接录入新论文 / 课题 / 专利））
 
# 论文著作与成果管理
## 数据导入（论文，专利，课题导入，支持批量数据导入、论文精选标记。批量导入的数据进行分页展示，可选择性进行精选标记，提供专利编辑（上传专利对应操作视频）（所上传的视频由卡片形式呈现，卡片背景是封面帧，封面帧可以根据上传视频直接生成，也要支持用户自己上传。页面有一个按钮点击跳转后⽀持系统自动压缩、生成封面帧，并⽀持 MP4 格式上传））
 
## 书籍仿真录入 （可以预留录入，允许用户除了书籍的基础信息，还可以上传封面、封底、核心章节图片。）

# 	自动化辅助功能
## 词云：（提取单篇论文关键词，展示所有收录论文关键词，支持人工设置关键词权重（权重越高，词云中字体越大）

## 图表生成：（后台需根据录⼊的年度数据，⾃动⽣成历年论⽂数量等统计折线图 。）

# 	活动管理
##  沉浸式编辑（供所见即所得的图文混排工具，支持活动内容（文字、图片、表格、视频等）的增删改，无需代码，排版与最终展示一致，可随时更新内容。（也要支持视频上传（所上传的视频由卡片形式呈现，卡片背景是封面帧，封面帧可以根据上传视频直接生成，也要支持用户自己上传。页面有一个按钮点击跳转后⽀持系统自动压缩、生成封面帧，并⽀持 MP4 格式上传），封面帧可以根据上传视频直接生成，也要支持用户自己上传。））
## 头条精选（就是给重要活动或新闻打「头条」标签，让它在前端的活动列表（卡片流）里置顶，相当于公众号的「头条文章」，用户一进页面就能看到。）
