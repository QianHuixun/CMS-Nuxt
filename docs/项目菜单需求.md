1. 系统基础配置管理
该部分负责大屏全局环境的维护，确保系统在不同负载下的性能表现。
导航配置：支持管理员自定义首页三个核心入口（资源导航、知识图谱、学术动态）的显示文字与副
标题 。
屏保管理：支持上传视频或图片集作为待机屏保，并设定无人触碰后的触发时间 。（可先做开屏页视
频、视频图片的设置）
检索词维护：支持配置大屏底部检索框内滚动的“重点研究热词” 。（这部分先不做）

2. 资源导航管理（一站式入口配置）
负责维护实验室各数据库及自研软件的上架与展示逻辑。
资源增删改查：提供标准化表单，录入资源名称、简介、封面图片、数据库访问链接或软件演示视频
。（封面图片不配置则为白底）
显示状态控制：为每个资源卡片设置“显示/隐藏”开关，便于在软件升级期间快速下架前端显示 。
拖拽排序管理：支持管理员在后台通过鼠标拖拽方式直接调整前端卡片的显示顺序 。（这个交互模式
可以修改，不一定拖拽）
分类管理：支持将资源划分为“数据库集群”与“自研分析工具”两个逻辑专区 。///（不需要）

3. 知识图谱展示控制（这部分可视化设置先保留，具体实现等待前端可视
化评审完成，可视化方式、交互前端参与调研）
由于图谱生成由底层系统完成，本后台主要负责前端的展示逻辑映射。
视觉映射配置：提供节点颜色与图标映射表，管理员可自定义不同实体类型（如药材、疾病、学者）
的视觉样式 。
物理引擎调节：提供“节点间距（斥力）”与“连线长度”调节杆，根据数据密度实时优化网络图的疏密程
度 。
接口配置：管理并维护图数据库（如 Neo4j）的实时读取接口 。

4. 学术动态内容管理
这是数据量最大的部分，涉及科研成果的结构化存储与关联。

4.1 人才与对应科研管理
专家库管理：录入学者个人简历、高清照片，支持照片裁剪 。
课题时空管理：支持按时间录入课题，管理课题级别、负责人及摘要 。
成果绑定组件：提供下拉菜单，实现学者与系统内已有的论文、专利数据一键关联 。

4.2 论文、著作与成果管理
批量数据导入：支持通过 Excel 等格式批量导入论文元数据（标题、作者、期刊等）。
论文精选标记：管理员可手动标记“高质量论文”，并上传 PDF 全文供前端全屏阅读 。
书籍仿真录入：支持批量上传书籍封面图及核心章节图片集，实现仿真翻页效果 。（？？？）
媒体自动处理：视频组件需支持系统自动压缩、生成封面帧，并支持 MP4 格式上传 。（专利演示视
频）

4.3 自动化辅助功能（词云）
关键词提取：在录入摘要时，后台系统需自动抓取并推荐关键词（如“天回医简”），支持人工修改权
重 。
统计图表生成：后台需根据录入的年度数据，自动生成历年论文数量等统计折线图 。

4.4. 活动剪影与内容编辑
沉浸式编辑：提供图文混排编辑工具，支持活动内容增删 。
头条精选：支持设置特定活动为“头条”，在卡片流中优先展示 。
