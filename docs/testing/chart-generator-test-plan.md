# 图表生成测试计划

## 功能范围

- 自动化辅助 > 图表生成
- 固定展示“历年论文数量统计”
- 数据来源为 `paper.publish_year`
- 后端接口 `GET /api/admin/paper/statistics/yearly-count`
- 前端展示 ECharts 折线图
- 图表下方展示“年份 / 论文数量”表格
- 支持刷新图表
- 支持空数据状态

## 不在本期范围

- 不做通用图表配置器
- 不做多图表配置
- 不做专利/课题图表
- 不做精选论文统计
- 不做导出图片/PDF
- 不做拖拽配置
- 不做全量 E2E

## 测试范围

- 后端按年份聚合论文数量
- `publish_year` 为空不计入
- `del_flag = '1'` 不计入
- 返回按年份升序
- 前端接口数据映射为图表和表格
- 空数据状态
- 刷新图表
- 回归 5.1 `paper/page`
- 回归 6.1 词云

## 验收标准

- 图表页能正常进入
- 有数据时展示折线图和表格
- 无数据时展示空状态
- 刷新按钮可重新拉取数据
- 统计数据来源于 `paper.publish_year`
- 不维护单独图表数据表
