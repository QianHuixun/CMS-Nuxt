# 头条精选测试计划

## 功能范围

- 后台活动可设置/取消头条
- `activity` 表 `is_headline` 字段落库
- 活动接口返回 `isHeadline`
- 活动列表支持 `isHeadline='1'` 查询
- 活动列表头条优先排序
- 头条精选页展示真实头条活动
- 头条精选页支持取消头条

## 不在本期范围

- 不新增前台活动列表页
- 不做全量 E2E
- 不做拖拽排序
- 不做定时上下头条
- 不做推荐算法

## 自动化测试范围

- `headline` 排序逻辑测试
- `isHeadline` 字段映射测试
- 头条查询逻辑测试
- 页面交互层组件测试

## 组件测试范围

- 后台活动列表/场景卡片能显示头条标签
- 设置活动为头条后，页面状态能更新
- 取消头条后，页面状态能更新
- 头条精选页能展示头条活动
- 头条精选页取消头条后，该活动从列表中移除
- 操作成功时有成功提示
- 操作失败时有错误提示或不错误更新本地状态

## 人工验收范围

- 后台设置头条
- 后台取消头条
- 场景卡片显示头条标签
- 头条精选页展示头条活动
- 取消头条后从头条精选页消失

## 验收标准

- `is_headline` 能正确落库
- `isHeadline` 能正确返回前端
- 头条活动能被查询出来
- 头条活动排序优先
- 后台主链路可正常操作
