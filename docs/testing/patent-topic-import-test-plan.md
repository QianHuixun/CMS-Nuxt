# 专利导入最小版 + 课题导入最小版测试计划

## 功能范围

- 数据导入页面新增论文 / 专利 / 课题三个 tab
- 论文 tab 保持原有能力
- 专利 tab 支持列表、查询、新增、编辑、删除
- 专利支持 MP4 操作视频上传
- 专利支持视频封面图保存与回显
- 专利视频以卡片形式展示
- 课题 tab 支持列表、查询、新增、编辑、删除

## 不在本期范围

- 不做真实 Excel/CSV 批量导入
- 不做专利视频后端压缩
- 不做后端自动抽帧
- 不做专利视频转码
- 不接入 topicSpaceTime 的时空字段
- 不做全量 E2E

## 测试范围

- 专利分页列表
- 专利名称查询
- 专利年份查询
- 专利新增
- 专利编辑
- 专利逻辑删除
- 专利视频 URL 保存与回显
- 专利封面 URL 保存与回显
- 课题分页列表
- 课题名称查询
- 课题年份查询
- 课题新增
- 课题编辑
- 课题逻辑删除
- 论文 tab 回归
- 6.1 词云回归
- 6.2 图表回归

## 验收标准

- 页面能看到论文 / 专利 / 课题三个 tab
- 专利主流程可用
- 课题主流程可用
- 专利视频和封面能保存并回显
- 论文、词云、图表不受影响
