# 4.1 人才管理测试计划

## 功能范围

- 人才管理菜单
- 专家库管理改名为人才管理
- 课题时空管理隐藏
- 人才分页列表
- 姓名 / 单位 / 研究方向 / 状态查询
- 新增人才
- 编辑人才
- 删除人才
- 上传高清照片
- 人才卡片背景展示照片
- 关联已有论文
- 关联已有专利
- 关联已有课题
- 详情展示人才信息和关联成果

## 不在本期范围

- 不做照片裁剪
- 不做课题时空管理
- 不在人才页面新增论文 / 专利 / 课题
- 不做简历 PDF 解析
- 不做复杂人才画像
- 不做全量 E2E

## 测试范围

- 人才增删改查
- 照片上传与回显

- 菜单显示与隐藏

- 6.1 词云回归
