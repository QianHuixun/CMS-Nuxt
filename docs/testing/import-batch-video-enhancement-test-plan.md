# 5.1 批量导入与专利视频增强测试计划

## 功能范围

- 论文 / 专利 / 课题三类 `.xlsx` 批量导入
- 三类导入模板下载
- 上传后解析预览
- 预览中展示错误行与错误原因
- 确认导入时只导入有效行
- 论文 tab 新增“只看精选”筛选
- 专利视频支持替换 / 删除
- 专利封面支持替换 / 删除

## 不在本期范围

- 不支持 `csv`
- 不做导入批次持久化
- 不做导入日志表
- 不做专利视频后端压缩
- 不做后端抽帧
- 不做转码
- 不做全量 E2E

## 测试范围

- 模板下载
- `.xlsx` 解析预览
- 必填字段校验
- 年份格式校验
- 有效行确认导入
- 论文精选筛选
- 专利视频替换 / 删除
- 专利封面替换 / 删除
- 论文 / 专利 / 课题增改删回归
- 6.1 词云回归
- 6.2 图表回归

## 验收标准

- 数据导入页可正常进入
- 三类模板都可下载
- 上传 `.xlsx` 后能先看到预览，不会直接入库
- 预览中能显示总行数、有效行数、错误行数、行号与错误原因
- 必填字段缺失和年份格式错误能被正确标记
- 确认导入后只写入有效行
- 论文 tab 的“只看精选”筛选可用
- 专利视频与封面可替换、删除并正常回显
- 论文 / 专利 / 课题原有增改删能力不受影响
- 6.1 词云与 6.2 图表功能不受影响
